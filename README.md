# The Photos

Um aplicativo para captura, armazenamento e visualização de fotos organizadas por localização e data.

---

## Tecnologias Utilizadas

* **React Native / Expo**
* **Expo Router**
* **Expo Image Picker**
* **Expo Location**
* **React Native Maps**
* **SQLite**

---

### Como Executar o Projeto

Instale as dependências:
   npm install

Inicie o Expo:
   npx expo start
